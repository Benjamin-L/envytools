=================
NV50:NVC0 signals
=================

.. contents::


Introduction
============

NV50 generation cards have the following counter domains:

- NV50:

  - 0: host clock
  - 1: core clock A
  - 2: core clock B
  - 3: shader clock
  - 4: memory clock

- NV84:NVC0 except NVAA, NVAC:

  - 0: host clock
  - 1: core clock A
  - 2: core clock B
  - 3: shader clock
  - 4: memory clock
  - 5: core clock C
  - 6: vdec clock
  - 7: core clock D

- NVAA, NVAC:

  - 0: host clock
  - 1: core clock A
  - 2: core clock B
  - 3: shader clock
  - 4: core clock C
  - 5: vdec clock
  - 6: core clock D

Shader clock
============

- 0x00-0x03: MPC GROUP 0
- 0x04-0x07: MPC GROUP 1
- 0x08-0x0b: MPC GROUP 2
- 0x0c-0x0f: MPC GROUP 3
- [XXX]
- 0x13: PCOUNTER USER_0 [NVA3:]
- 0x14: PCOUNTER USER_1 [NVA3:]
- 0x20: trailer base
